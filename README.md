## evaluacion_js

**Funciones:**
- **Funcion comprobarPresupuesto()**: Verifica si el numero ingresado por el usuario sea mayor a 700000 o menor, si este es mayor, hace un descuento del 15% si es menor hace un descuento de 5%, esta información la obtiene de la funcion ejercicio_1

- **Funcion ejercicio_1()**: Es el que se encarga de obtener los datos ingresados por el usuario, y se conecta con la funcion comprobarPresupuesto, verifica que los datos ingresados sean los solicitados por la primera funcion (No este vacio), Este muestra los datos por el container.

- **Funcion agregarLista()**:  Obtiene los datos ingresados en la funcion ejercicio_2_1(), este los toma y los manda al final del arreglo.

- **Funcion ejercicio_2_1()**: Obtiene los datos ingresados por el usuario, estos datos lo manda a la funcion agregarLista(), Este muestra los datos por el container.

- **Funcion agregarUrgencia()**: Obtiene los datos ingresados en la funcion ejercicio_2_2(), este los toma y los manda al inicio del arreglo.

- **Funcion ejercicio_2_2**: obtiene los datos ingresados por el usuario, estos datos los manda a la funcion agregarUrgencia(), Este muestra los datos por el container.

- **Funcion buscarDatos()**: Obtiene los datos ingresados (rut o nombre) de la funcion ejercicio_3(), Esta funcion verifica si uno de los datos ingresados pertenecen a algunos de los 2 arreglos (rut, nombre3), si estos datos no estan en la lista se mostrara un mensaje el cual solicitara uno de los 2 datos.

- **funcion ejercicio_3**: Obtiene los datos ingresados por el usuario y los conecta a la funcion buscarDatos(), Este muestra los datos por el container.
